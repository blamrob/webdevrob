Hotel Zelos

http://www.hotelzelos.com/

I was responsible for the front-end build of this site. This is a fully responsive website with interesting CSS transitions and animation. The design was adapted from Photoshop files created by our designer, but the various responsive states needed to be created during development. I worked with the designer, the client and our account managers to find the best solution to these challenges.

This site became a template for other hotels owned by this management company. Each property would get a light reskin of this design to fit their style.

Respond

http://userespond.com/    ( Screen Shots )

This is a web application for hotels to book and process meeting proposals. I was responsible for the front-end build of this application. It’s fully responsive. I also created a custom font-based icon set to use. Many of the initial icons were SVGs obtained through The Noun Project, but I had a hand in cleaning up each the vector files. Some were made a little more simple to read better at smaller sizes, some were mixed and matched to create new ones, and a few were entirely new.

Experience

http://useexperience.com/

This is a web based mobile app for guests who check in to a hotel to get more information about it. It has important phone numbers they can click to dial, information on things to do and places to eat. Both on property and locally. I was responsible for the initial static build of this application. While not fully responsive, this site is intended to be used by guests on their phone.

An updated fully responsive version of this site is in progress.
